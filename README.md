# lily-ui
an ui lib based on reactjs

- 使用 coffee, scss 编写实现
- 使用 gulp 编译打包

# 编译方法

工程使用 gulp 组织源代码和进行编译。  
系统内需要安装 node.js  

```
npm install -g gulp # 安装 gulp
npm install # 安装环境依赖
gulp watch # 启动编译进程
```

# 目录组织

```
-- src
    -- base
        -- coffee
          -> color.coffee
        -- secc
          -> color.scss

```
